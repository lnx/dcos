简介

为了更好的展示论文中相关算法的具体执行效果，开发了一套可交互的软件对论文中重点章节的研究内容进行展示。通过展示，可以对论文中设计的协同观测系统有更清晰的了解，同时也能看到算法执行中的一些细节。

命令说明

本部分将对系统中的控制命令给出说明。

命令 参数 说明
CLEAR NONE 清除本次所有控制命令
INIT -1 | num1, num2, num3, ... 系统初始化
AGENT NONE 开始选举区域代理节点
TENURE number 设定区域代理任期
START NONE 开启监测模块
STOP NONE 关闭监测模块
SLEEP NONE 使用半睡眠调度
EVENT CLEAR | latitude longitude 事件清除或者添加新的事件
KILL sid 停止某个传感器节点
